标准浮点类型

C++ 类型字节数指数位尾数位动态范围说明
float4823±3.4×10³⁸IEEE 754 单精度
half（__fp16）2510±65504IEEE 754 半精度
bfloat16_t287±3.4×10³⁸Brain Float 16，动态范围同 FP32

A5 新增类型

C++ 类型字节数说明
float8_t（FP8 E4M3）14 位指数 + 3 位尾数，精度优先，范围 ±448
float8_e5m2_t（FP8 E5M2）15 位指数 + 2 位尾数，范围优先，范围 ±57344
mxfp4_t（MXFP4）0.5Microscaling FP4，4-bit 超低精度，用于极致压缩
hif8_t（HiF8）1华为自定义 8-bit 浮点，可配置指数/尾数分配

这些类型主要面向大模型推理场景的量化需求。A5 的 Vector 单元原生支持 FP8 和 MXFP 的计算指令。

对齐粒度速算

32 字节对齐下，各数据类型每次搬运的最少元素个数：

数据类型字节数对齐粒度（元素数）计算方式
FP324832 / 4 = 8
FP16 / BF1621632 / 2 = 16
FP813232 / 1 = 32
INT648432 / 8 = 4
INT324832 / 4 = 8
INT1621632 / 2 = 16
INT813232 / 1 = 32

sswap 处理的是 FP32 数据，所以 ELEMENTS_PER_BLOCK = 32 / 4 = 8，每次 DataCopy 搬运的元素个数必须是 8 的倍数。详细对齐规则见 附录 B：对齐规则速查。
